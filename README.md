# 낙서몬 아파트(Oddyroom)

## 1. 프로그램 개요

낙서몬 아파트는 Flutter를 활용하여 제작한 캐릭터 상호작용 앱입니다.

사용자는 여러 방에 배치된 캐릭터들을 관찰할 수 있으며, 캐릭터들은 랜덤으로 움직이거나 감정 상태에 따라 메시지를 출력합니다.

단순한 기능 구현을 넘어, 귀엽고 친근한 캐릭터 브랜딩을 통해 사용자가 반복적으로 확인하고 싶어지는 앱을 목표로 제작하였습니다.

## 2. 주요 기능

- 여러 방으로 구성된 아파트 화면
- 방마다 캐릭터 배치 기능
- 캐릭터 랜덤 이동 기능
- 캐릭터별 성격 메시지 출력
- 다른 캐릭터가 방문하는 이벤트
- 상태에 따른 말풍선 메시지 출력
- 귀여운 캐릭터 중심의 브랜딩/UI 구성
